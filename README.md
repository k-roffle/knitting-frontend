# Knitting-frontend

`Knitting`의 프론트엔드 프로젝트입니다.

## 프로젝트 세팅


### git hooks 설정

```bash
$ git config core.hooksPath .githooks
```

명령어를 입력해 프로젝트 경로를 Git Hooks 경로로 만들어줍니다.

```bash
$ git config --get core.hooksPath
```

명령어를 입력했을 때 `.githooks`를 가르켜야 합니다.

### 패키지 설치

```bash
$ yarn
```

### 서버 실행

```bash
$ yarn start
```
